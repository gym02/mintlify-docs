## 六、前端

### 1、React 生命周期（以函数组件为主）

- 挂载：首次渲染；`useEffect(() => {}, [])` 在挂载后执行。  
- 更新：state/props 变化触发重新渲染；useEffect 依赖变化后执行。  
- 卸载：组件移除；useEffect 返回的 cleanup 函数在卸载时执行。  
（类组件：componentDidMount / componentDidUpdate / componentWillUnmount）

### 2、什么是虚拟 DOM

虚拟 DOM 是对真实 DOM 的一层 JS 对象抽象。React 通过对比新旧虚拟 DOM（diff），计算出最小变更，再批量更新真实 DOM，从而减少直接操作 DOM 的成本，并带来更可预测的 UI 更新模型。

### 3、Vue3 生命周期（面试好背版）

- onMounted：组件第一次渲染到页面后执行，用来做初始化（请求数据、绑定事件、开定时器、初始化图表等）。  
- onUpdated：组件更新并完成 DOM 更新后执行；如果要拿到更新后的 DOM，配合 nextTick。  
- onUnmounted：组件销毁时做清理（移除事件监听、清定时器、关 WebSocket），避免内存泄漏和重复执行。  

一句话总结：挂载做初始化，更新后拿 DOM 用 nextTick，卸载做清理。
