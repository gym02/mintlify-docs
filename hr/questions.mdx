## 人事常见问题

### 1、你为什么离职？（面试口述参考）

**参考回答（按自己语气说即可）：**

我上一家公司之前是**双休**，工作节奏相对规律。后来公司调整制度，**周六也需要上班**，整体强度变大，**身体上和时间安排上都会更累一些**。

从个人规划上，我希望在一个**更稳定、节奏更可预期**的平台上长期发展，把精力持续放在业务和技术成长上，而不是被频繁变化的作息牵制。我了解到**贵公司**在平台和团队稳定性方面口碑不错，也和我的期望比较契合，所以希望有机会加入，**能够长期稳定地发展下去**。

**表达时注意：**

- **少抱怨前公司**，多讲**客观变化**（制度调整）和**自己的职业诉求**（稳定、长期发展）。  
- **「贵公司」** 面试前换成对方公司名或「咱们团队」，显得更自然。  
- 若面试官追问「能适应加班吗」，可补一句：能接受**项目需要时的合理加班**，更看重**节奏是否可持续、团队目标是否清晰**。

**面试一句话：** 前司从双休改为单休、强度上升；我希望在**更稳定的平台长期发展**，认可贵司方向，所以投递。

### 2、你未来的职业发展规划是什么？（面试口述参考）

**参考回答（技术岗通用结构：近 → 中 → 长期 + 与岗位结合）：**

**短期（1～2 年）**：先把岗位上的技术栈和业务摸熟，能在团队里**独立、高质量地完成需求**，代码规范、联调、线上问题都能扛得住；同时把**基本功**（比如 Java / 中间件 / 你们栈里常用的部分）打扎实。

**中期（3～5 年）**：我会深入去学习微服务，高并发，分布式等这些技术。

**长期（5 年及以后）**：长期来看的话，我希望能够成为技术骨干，在技术上持续成长，在公司能长期发展

自己平时也有关注和了解AI相关的知识，未来的话也会持续的学习这方面的内容。
