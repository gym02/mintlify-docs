## 一、深度学习基础

（1）**什么是深度学习？和机器学习的区别？**

深度学习是机器学习的一种，核心是使用多层神经网络自动学习数据的高层特征。与传统机器学习相比，深度学习减少了人工特征设计，通过端到端训练直接从原始数据中学习特征表示。

---

**补充：常见概念有哪些？深度学习、机器学习、神经网络三者的关系？什么是数据的高层特征？**

**常见概念**：数据、特征、标签；训练集/验证集/测试集；模型、参数；损失函数；优化器；梯度与反向传播；过拟合/欠拟合；正则化；学习率；Batch；归一化；推理。

**三者关系**：

- **机器学习**：让计算机从数据里自动学出规律、用来做预测或决策的一类方法的总称。不限定具体用什么模型（可以是线性回归、决策树、神经网络等）。
- **神经网络**：一种具体的模型结构，本质是一个“可学习的函数” y=f(x)，由多层“线性变换 + 激活函数”叠成，参数（权重、偏置）通过训练数据自动学出来。
- **深度学习**：特指“用层数很多（很深）的神经网络”来做机器学习的那一块。所以：深度学习是机器学习的一种，神经网络是深度学习最常用的模型；可以简单记成“机器学习 ⊃ 深度学习，深度学习用神经网络”。

**高层特征**：更“抽象/有语义”的信息。比如图像里从边缘纹理（低层）组合成形状部件（中层），最后变成“这是一只猫/一辆车”（高层）。

（2）**神经网络训练的目标是什么？**

通过最小化损失函数，使模型预测结果尽可能接近真实标签，从而学习到数据的内在规律。

---

**补充：什么是“最小化损失函数”和“真实标签”？**

**真实标签**：训练数据里给定的“标准答案”，比如图片对应的类别（猫/狗）、房价的真实数值、文本的情感（正/负）。

**最小化损失函数**：先用模型算出预测结果，再用损失函数衡量“预测和真实标签差多少”，训练就是不断调整模型参数，让这个差距尽量变小。

**补充：神经网络是如何调整模型参数的？是自己调还是人手动调？**

训练时参数是程序自动更新：先前向计算得到预测和损失，再用反向传播计算每个参数对损失的影响（梯度），最后由优化器（如 SGD/Adam）按学习率把参数往“让损失变小”的方向更新。

人主要做的是设置训练配置和超参数（比如网络结构、学习率、batch size、训练轮数等），以及准备数据和选择损失函数/优化器；真正的参数更新过程由训练算法自动完成。

（3）**为什么神经网络需要激活函数？**

如果没有激活函数，神经网络本质上只是多个线性变换的叠加，等价于一个线性模型，无法拟合非线性关系，激活函数引入非线性，使网络具备更强的表达能力。

---

**补充：激活函数是什么？**

激活函数是什么？

激活函数就是加在每一层“线性计算”后面的非线性函数。它把神经元的线性输出 \(z\) 变成 \(a\)，让网络能表示非线性关系。

神经元计算：先线性，再激活

（公式）z = w^T*x + b，a = σ(z)

其中 σ(·) 就是激活函数（如 ReLU、Sigmoid、Tanh）。

为什么需要它：如果没有激活函数，每层都是线性变换，多层叠加仍然等价于一个线性变换，模型表达能力就很弱。

常见激活函数例子：

- ReLU：σ(z)=max(0, z)
- Sigmoid：σ(z)=1/(1+e^(-z))
- Tanh：σ(z)=tanh(z)

**神经网络架构长什么样？（以最简单的 MLP（多层感知机） 为例）**

结构：输入层 → 若干隐藏层 → 输出层

每一层做的事：x → 线性 Wx+b → 激活 σ(·)

隐藏层 = 它包含了多个：线性层 和 对应的激活函数

示意图（全连接网络）：

```
输入 x  →  [线性: W1x+b1] → [激活 σ] →  h1
        →  [线性: W2h1+b2] → [激活 σ] →  h2
        →  [线性: W3h2+b3] →  输出 ŷ
```

一句话总结：神经网络就是把“线性变换 + 激活函数”重复堆叠起来的可学习函数 y=f(x)。

（4）**常见激活函数及优缺点？**

Sigmoid：输出在（0,1），但容易梯度消失

Tanh：输出在（-1,1），且以 0 为中心，但仍有梯度消失问题

ReLU：把负数变成 0，正数原样通过。计算简单、收敛快，缓解梯度消失，但可能出现神经元死亡

---

**补充：这三个激活函数在做什么？**

可以把它们理解成同一件事：先做线性计算得到一个数 z（比如 z=Wx+b），再用激活函数把 z “变形”成输出 a=σ(z)。因为 σ 不是直线函数，所以网络就具备了非线性表达能力。

Sigmoid：把任意 z 压到 0 ~ 1，更像“概率”；z 很大/很小时曲线会变平，学习信号变弱，所以容易梯度消失。  
Tanh：把任意 z 压到 -1 ~ 1，且以 0 为中心；同样在两端会变平，所以也可能梯度消失。  
ReLU：把负数变成 0，正数原样通过（a=max(0,z)）；计算简单、收敛快，但如果一个神经元长期输出为负，就可能一直输出 0（“神经元死亡”）。

直观例子（输入 z，输出大概是多少）：

| z | Sigmoid(z) | Tanh(z) | ReLU(z) |
|---|------------|---------|---------|
| -3 | ≈0.05 | ≈-1.00 | 0 |
| -1 | ≈0.27 | ≈-0.76 | 0 |
| 0  | 0.50 | 0.00 | 0 |
| 1  | ≈0.73 | ≈0.76 | 1 |
| 3  | ≈0.95 | ≈1.00 | 3 |

**梯度是什么？**

梯度就是“损失函数对参数的变化率”。直观理解：参数往哪个方向改、改多少，损失会下降最快。训练时优化器就是靠梯度来更新参数（例如“参数减去学习率×梯度”）。

**梯度消失是什么？**

梯度消失就是：在反向传播时，传到前面层的梯度变得非常小（接近 0），导致这些层的参数几乎不更新，训练变慢甚至学不动。

**为什么会消失（直觉版）**

反向传播会把梯度一层层传回去，本质上会出现多项相乘；如果其中很多因子长期接近 0 或小于 1（比如 Sigmoid/Tanh 在两端很平，导数很小），乘多了就越来越小，前面层收到的梯度就接近 0（梯度消失）。

**反向传播是什么？**

反向传播（Backprop）就是：在你算完预测和损失 L 之后，从输出层往前一层层把梯度算出来，最终得到每个参数的相关的偏导数，让优化器知道“该怎么改参数”。

一句话：前向算结果，反向算梯度。

**学习率是什么？**

学习率（learning rate）就是每次更新参数时的“步子大小”。

- 步子太大：可能越更新越偏、震荡甚至发散
- 步子太小：能收敛但很慢

训练里常见更新形式（直觉版）：

`θ ← θ − η ⋅ ∇θ L`

其中 η 就是学习率。

**“导数连乘”里说的导数是什么？**

这里的“导数”指的是链式法则里每一层贡献的那一项，例如：

- 激活函数的导数：∂a/∂z（输出 a=σ(z) 对输入 z 的变化率）
- 线性层的导数：∂z/∂W、∂z/∂x

反向传播本质是用链式法则把“最终损失对前面某个量的影响”拆成一串乘积，比如：

`∂L/∂x = (∂L/∂a) ⋅ (∂a/∂z) ⋅ (∂z/∂x)`

层数越多，这种“每层一项”的乘积越长；如果很多项都小于 1 或接近 0，就容易越乘越小（梯度消失）。

**二、卷积神经网络（CNN）**

（1）**CNN为什么适合图像任务？**

CNN能利用图像的局部相关性和平移不变性，通过局部连接和权值共享大幅减少参数量，提高特征提取能力。

**补充说明**

- **局部相关性**：图像里相邻的像素往往是一块一块有关系的（比如一块是眼睛、一块是边缘），不需要把整张图一次性全连起来看，用一个小窗口（卷积核）在局部滑动就能抓住这些模式。
- **平移不变性**：同一种 pattern（比如“竖边”）出现在图里不同位置，用同一套卷积核去扫，在哪都能认出来，所以不需要为每个位置学一套新参数。
- **局部连接**：每个神经元只和输入的一小块区域（感受野）相连，而不是和整张图相连，参数量少很多。
- **权值共享**：同一个卷积核在整个图上滑动、到处复用，参数只和卷积核大小、通道数有关，和图像尺寸无关，所以大图也不会暴增参数。

**典型 CNN 架构**：输入层 → 多个个「卷积块」堆叠 → 展平特征 / 全局池化 → 一两层全连接（输出分类/回归结果）。
**CNN 大致流程**：输入图像 → 若干“卷积层（提取特征）+ 激活 + 池化（缩小尺寸）”重复几轮 → 特征图拉平 → 全连接层 → 输出（分类概率等）。前面几层学到边缘、纹理，越往后越抽象（部件、物体），最后用全连接做决策。

（2）**卷积层的作用是什么？**

卷积层通过卷积核在局部区域滑动提取特征，如边缘、纹理等，并逐层学习更高层语义特征。

（3）**池化层的作用？**

池化层用于降低特征图分辨率，减少计算量，同时增强模型对小幅平移的鲁棒性。

（4）**卷积为什么能减少参数？**

因为卷积用的是“局部扫描 + 权值共享”，同一卷积核在整个特征图上滑动，因此参数数量只与卷积核大小和通道数有关，与图片输入尺寸大小无关。

（5）**什么是感受野？**

感受野是指卷积网络中某个神经元在原始输入上能“看到”的那块区域范围，随着卷积和池化层叠加，感受野会越来越大。

（6）**1×1卷积有什么作用？**

用于改变通道数、融合特征，并减少计算量。

1×1 卷积在每个像素点上对通道做线性变换，相当于“只改通道，不看邻居”，用来调节通道数、融合不同通道的特征，并配合其他卷积一起减少计算量。

**三、ResNet（残差网络）**

（1）**ResNet解决了什么问题？**

ResNet主要解决了深层网络中梯度消失和网络退化问题，使网络可以安全地加深。

补充：ResNet 中文通常叫**残差网络**，也常写作“**深度残差网络**”。

（2）**网络退化问题是什么？**

随着网络加深，训练误差反而上升，说明模型难以优化而非表达能力不足。

（3）**ResNet一定比普通CNN好吗？**

不一定，在浅层网络或简单任务中优势不明显，甚至可能增加计算开销。

（4）**什么是残差结构？**

残差结构通过引入捷径连接，让网络学习残差映射而不是直接学习完整映射H(x)。

补充：三个关键词怎么理解？

1）捷径连接（shortcut connection）是什么？
捷径连接就是“跨过若干层”，把输入 x 直接绕行到残差块的后面，与中间计算得到的结果相加（或拼接）。这样信息和梯度有一条更短的路可以直接传回前面层。

2）残差映射（residual mapping）是什么？
残差映射可以理解为需要额外“补偿”的那部分变化，用 F(x) 表示；残差块的输出通常是：输出 = F(x) + x。

3）完整映射（full mapping）是什么？
完整映射 H(x) 指的是“从输入 x 到目标输出”的整段变换。如果没有捷径连接，网络就得直接把整个变换 H(x) 学出来；而引入残差结构后，网络改为学习“相对于 x 还差多少”，也就是学习 F(x)。

（5）**为什么残差结构有助于训练深层网络？**

残差连接提供了一条梯度直接传播的路径，缓解梯度消失，使深层网络更容易优化。

**四、RNN（循环神经网络）**

（1）**RNN适合处理什么任务？**

RNN适合处理序列数据，如时间序列、文本、语音等，因为它能利用历史信息。

补充：RNN 是什么？

RNN（循环神经网络）是一类专门处理序列数据的神经网络，它在每个时间步都会接收当前输入 \(x_t\)，并且把上一时刻的隐藏状态 h<sub>t-1</sub> 一起作为信息输入到当前时刻，从而得到新的隐藏状态 \(h_t\) 和输出。

可以把它记成一句话：**“同一套计算在时间维度上循环使用”，让模型带着“记忆”（隐藏状态）往后处理。**

（2）**RNN的“记忆”是如何实现的？**

通过隐藏状态h，当前时刻的输出依赖于当前输入和上一时刻的隐藏状态。

补充：隐藏状态 h 是什么？

隐藏状态 h 可以理解为“模型在当前时间点保存下来的信息摘要”，用来把过去已经看过的内容压缩成一份向量表示；下一时刻会继续使用这份向量，所以它就像记忆。

补充：隐藏状态是怎么计算出来的？

RNN 会把当前输入 \(x_t\) 和上一时刻的隐藏状态 h<sub>t-1</sub> 送进同一个 RNN 计算单元，得到新的隐藏状态 \(h_t\)（并可选地产生输出 \(y_t\)）。

常见写法（直觉版）：

- h_t = f(Wx * x_t + Wh * h<sub>t-1</sub> + b)
- y_t = g(Wy * h_t + by)  （如果当前任务需要输出）

其中 f 和 g 通常是某种激活函数（比如 tanh 或 ReLU）。

补充：RNN 的架构长什么样？

可以把它想成“同一个小模块在时间轴上反复使用”（权重在不同时间步共享），但**计算顺序依赖时间**：第 t 步需要用到上一时刻的隐藏状态 h<sub>t-1</sub>，所以不是并行的。

```
x1 -> RNN 单元 -> h1 -> y1
x2 -> RNN 单元 -> h2 -> y2
x3 -> RNN 单元 -> h3 -> y3
...
```

理解点：把它叫做“展开后的 RNN”（unrolling），展开图里看起来像多个小模块，但它们是同一套参数在不同时间步被重复调用；同时 h 会从上一时刻传到下一时刻，保证序列依赖。

每一步都会用到上一时刻的 h，所以信息沿着时间顺序被带下去。

（3）**RNN的主要问题？**

在长序列中容易出现梯度消失或梯度爆炸，难以建模长期依赖。

补充：什么是梯度爆炸？

梯度爆炸是指在反向传播时，梯度在多层网络/多时间步之间用链式法则不断“被放大”，最终变得非常大（甚至数值不稳定）。结果是参数更新幅度失控，训练容易震荡、loss 变大，严重时会直接发散（无法收敛）。

**五、LSTM（长短期记忆网络）**

（1）**LSTM相比RNN的改进？**

LSTM引入了门控机制和细胞状态，可以有选择地记住或遗忘信息，从而缓解梯度消失问题。

**补充：那是否能解决梯度爆炸问题呢？**

梯度爆炸也可能被缓解，但通常不是“LSTM 100% 直接解决梯度爆炸”。梯度爆炸在 RNN 里更常见的处理方式还包括：梯度裁剪（gradient clipping）、更合适的初始化和优化器等。

（2）**LSTM有哪些门？作用是什么？**

遗忘门：决定丢弃多少旧信息  
输入门：决定写入多少新信息  
输出门：决定输出多少当前状态信息

（3）**LSTM为什么能记住长期依赖？**

细胞状态提供了一条相对稳定的信息传递路径，使梯度可以在时间维度上传播得更远。

**六、Transformer（变换器）**

（1）**Transformer为什么不使用RNN？**

Transformer使用自注意力机制，可以并行计算，避免RNN的顺序依赖，训练效率更高。

**补充：Transformer 的大致架构是什么？**

经典 Transformer 架构分为两部分：编码器（Encoder）和解码器（Decoder）。

- 编码器：负责“理解输入”，是**双向注意力**，每层通常包含多头自注意力 + 前馈网络（FFN）+ 残差连接 + LayerNorm。是**双向注意力**
- 解码器：负责“生成输出”，是**单向注意力**，每层通常包含 Masked 自注意力 + 编码器-解码器注意力（Cross-Attention）+ 前馈网络（FFN）+ 残差连接 + LayerNorm。

**补充：BERT 和 ChatGPT 分别对应什么方向？**

- BERT：Encoder-only（只有编码器），是**双向注意力**（同时利用左、右上下文），更偏“理解”任务，典型场景包括文本分类、语义匹配、信息抽取、语义表示提取等。
- GPT/ChatGPT：Decoder-only（只有解码器），是**单向注意力**（按从左到右逐 token 生成）；解码器里有 **Masked Self-Attention（掩码注意力）**，用于屏蔽未来 token，避免“偷看答案”。

一句话记忆：原始 Transformer 是“编码器+解码器”；BERT 是“双向编码器，偏理解”；GPT/ChatGPT 是“单向解码器（带掩码），偏生成”。

（2）**Q、K、V的含义？**

Q（Query）：当前词想关注什么  
K(Key)：每个词提供的索引信息  
V(Value）：真正参与加权求和的信息

补充：
Q（Query，查询）：我现在这个词/位置“想找什么信息”
K（Key，键）：每个词/位置“我这里有什么类型的信息，可不可以被你匹配”
V（Value，值）：真正要被取走并加权汇总的内容

（3）**为什么需要位置编码？**

因为Transformer本身不包含序列顺序信息，位置编码用于向模型注入位置信息。

（4）**什么是自注意力机制？**

自注意力通过计算序列中各位置之间的相关性，使模型在生成表示时能够关注到全局信息。

补充（详细易懂版）：

- **“自”是什么意思**：序列里的每个位置（每个词/每个 token）都会去**看一遍序列里所有位置**（包括自己），用注意力分数决定“谁更重要”，所以叫自注意力（Self-Attention）。
- **大致在干什么**：对每个位置，用它的 Q 去和所有位置的 K 算匹配度，得到一组权重；再用这些权重对所有位置的 V 做加权求和，得到该位置的新表示。这样当前位置的向量里就**融合进了全序列的信息**。
- **和 RNN 的差别**：RNN 是一步步把信息往后传；自注意力是**一步里让每个位置直接和任意位置建立联系**，更容易捕捉长距离依赖，也更容易并行计算（不依赖上一步算完才能算下一步）。

（5）**多头注意力的好处？**

多头注意力可以让模型从多个子空间关注不同语义关系，提升表示能力。

补充（详细易懂版）：

- **单头在干什么**：一次注意力只能学出一种“看谁、看多重”的模式（例如更关注相邻词，或更关注主谓关系）。
- **多头在干什么**：把维度拆成多份，**并行跑多组独立的 Q、K、V**，每组算一套注意力，再在特征维度上拼起来。相当于多个“小专家”同时看同一句话，各自关注不同侧面。
- **为什么有用**：有的头可能偏重语法（主谓一致），有的头可能偏重指代（代词指谁），有的头可能偏重局部搭配。合在一起，**表达能力更强**，不容易被一种固定模式绑死。

（6）**为什么要除以√d？**

防止点积过大导致softmax梯度过小，保持训练稳定。

补充（详细易懂版）：

- **d 是什么**：这里一般指每个头里 Q、K 向量的维度（记作 d，有的书写 d_k）。
- **点积会变大**：Q 和 K 做点积时，维度越高，各项相乘再相加，**数值往往整体更大**（随机情况下大致和维度的平方根成比例变大）。
- **大了会怎样**：点积特别大时，经过 softmax 会**几乎变成“只选一个位置为 1、其余接近 0”**，梯度很小，不好训练；而且数值尺度随 d 变化也不稳定。
- **除以根号 d 的作用**：把点积**缩放到一个相对稳定的范围**，让 softmax 不会那么容易饱和，训练更稳。可以把它理解成一种**按维度做归一化**的技巧。

（7）**Transformer如何建模长距离依赖？**

任意位置之间可直接计算注意力，不受距离限制。

补充（详细易懂版）：

- **什么是长距离依赖**：序列里**相距很远**的两个位置，在语义上仍然要互相依赖。例如：句首的主语和句末的谓语要一致；前面出现的“小明”和后面出现的“他”要对应；否定词和它所否定的内容要挂钩。距离越远，模型越难把两边信息**可靠地联系起来**，这就叫需要建模**长距离依赖**。
- **和“近处依赖”对比**：相邻词的搭配（如形容词修饰名词）往往容易学；**跨很多词**仍要用的信息，才是长距离依赖的典型场景。
- **如何理解本题的答案**：自注意力在算当前位置时，会**同时看到序列里每一个位置**（用 Q、K、V 算权重再聚合），不会因为两个词中间隔了很多词就“多绕几步路”。所以从机制上讲，**第 1 个词和第 100 个词之间也是一步直连**，不像 RNN 那样信息要沿时间步一层层传过去，因此更容易捕捉远距离关系。（多层堆叠后，表达能力还会进一步加强。）

（8）**Transformer和RNN的本质区别？**

RNN依赖时间递归，Transformer依赖全局注意力和并行计算。

**七、多层感知机**

（1）**什么是多层感知机？**

多层感知机是由输入层、一个或多个隐藏层和输出层组成的前馈神经网络，层与层之间采用全连接结构。

（2）**MLP能解决什么问题？**

MLP能拟合复杂的非线性函数，适合处理结构化数据或低维特征数据，如表格数据、特征向量等。

（3）**MLP的缺点是什么？**

参数量大、容易过拟合，且无法利用数据的空间或时序结构，不适合直接处理图像或序列数据。
