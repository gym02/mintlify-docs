## OpenClaw 是什么

**OpenClaw**（大家常叫「龙虾」）是一个**开源、本地优先的 AI 智能体**，定位不是聊天机器人，而是**能真正执行任务的数字助手**。

**一句话总结**：通过接入微信、飞书、钉钉、企业微信等**第三方社交/办公平台**，你用语音或文字与**本地（或自己服务器）部署的智能体**对话，智能体在你这台机器或服务器上**实际执行**你规划的任务——例如收邮件、整理文件、跑命令、查代码、做自动化——并把结果通过同一平台反馈给你。数据留在本机或自建环境，不经过第三方，适合既要「能干活」又在意隐私和可控性的个人或团队。

---

## 能干什么（核心能力）

### 文件和系统

- 批量重命名、整理、备份、分类文件
- 读写编辑文档、清理冗余、压缩图片
- 执行终端命令、查看系统状态

### 浏览器自动化

- 自动搜索、填表、登录、点击、翻页
- 网页数据抓取、页面监控、批量下载

### 办公流程

- **邮件**：筛选、自动回复、批量发送
- **日程**：同步日历、提醒、行程规划
- **文档**：会议纪要、周报等生成与整理

### 代码与开发

- 生成代码、调试、写测试
- PR 合并、CI/CD 等开发流程辅助

### 多平台对话接入

- 企业微信、飞书、钉钉、Slack、Discord、Telegram 等
- 微信公众号、微信小程序
- 支持私聊和群聊，从对话里触发任务

## 技术特点（和别家比有啥不一样）

| 点 | 说明 |
|----|------|
| **本地/自托管** | 跑在自己电脑或服务器，数据不出本机，可完全自管 |
| **模型灵活** | 可接 Claude、GPT、豆包、智谱等云端模型，也可接本地模型 |
| **技能扩展** | 有大量预置技能，也支持自己写技能扩展能力 |
| **低延迟** | 通过异步请求池等技术，模型调用延迟可压到约 380ms 量级 |

## 怎么用

- **安装**：支持 macOS、Windows、Linux，官方提供一行命令安装脚本，也会自动装 Node 等依赖。
- **初始化**：安装后执行 `openclaw onboard` 做首次引导，按提示配好模型和要用的通道（如企业微信、飞书等）。
- **入口**：可以在终端用 CLI，也可以用桌面助手（如 macOS 菜单栏）配合使用；若接了企业微信/飞书/钉钉等，直接在对应 App 里对话即可让 OpenClaw 干活。

**官方**：[OpenClaw 官网](https://openclaws.io/zh/) · [中文文档](https://openclaw.cc/) · [GitHub](https://github.com/openclaw/openclaw)

## 别人都拿他干什么了——具体使用例子

下面是从官方文档、社区（如 Moltbook）和用户分享里整理出的**真实用法**，按场景分类，方便你照着试或改一改给自己用。

### 个人效率

**每日 AI 新闻简报**  
在 Telegram / 飞书 / 钉钉里对 OpenClaw 说：「每天早上 8 点给我推送一份 AI 领域的新闻简报，来源包括 TechCrunch、Hacker News」。它会用 Browser Skill 抓取、总结，并创建定时任务（Heartbeat），到点自动推送，不用再自己刷资讯。

**待办 + 日历**  
「@OpenClaw 添加待办：明天下午 3 点开会讨论项目进度」「查看今天的待办」「完成任务：写周报」。配合 Todoist 等 Skill，用自然语言管待办，自动同步到你的任务列表。

**邮件分类与提醒**  
「帮我监控邮箱，如果收到包含『紧急』或『重要』的邮件，立即通知我」。装好 Email Skill 并配置账号后，重要邮件会推到你绑定的飞书/钉钉/Telegram，避免漏看。

**个人知识库**  
把文章链接或本地笔记目录丢给 OpenClaw，它自动抓取全文、OCR 识别 PDF，做成可检索的知识库。之后直接问「搜索我的笔记：关于 Docker 的使用技巧」「总结我关于 React 的所有笔记」，不用再翻收藏夹。

**本地文件整理**  
「帮我归类文档」「扫描这个文件夹里的报销发票，生成报销表格」「清理磁盘」。借助系统权限和对应 Skill，在飞书里远程发指令，OpenClaw 在本机完成整理、OCR、生成表格，全程可无人参与。

### 团队协作

**项目状态 + GitHub**  
在飞书群：「@OpenClaw 查看项目 xxx 的最新进度」「列出所有未关闭的 Issues」「创建 Issue：添加多语言支持」。装 GitHub Skill 并配置 Token 后，不用切网页就能查进度、建 Issue。

**会议记录与待办**  
开会时在群聊里 @OpenClaw「开始记录会议」；散会后说「总结会议内容，提取待办事项」。它会从聊天记录里提取关键决策和行动项，自动创建待办，并生成会议纪要，避免「开完会就忘」。

**飞书/钉钉群里的「数字员工」**  
有人用 OpenClaw 管群：配置权限和安全规则后，让它负责新人入群指引、定时发内容、维持群内秩序，相当于一个 7×24 在线的群管。

### 开发与运维

**GitHub PR 自动审查**  
「审查这个 PR：https://github.com/xxx/pull/123」或「监控仓库 xxx 的新 PR，自动进行代码审查」。OpenClaw 会分析测试覆盖率、安全问题、性能建议、潜在 bug、代码风格，把结果发到指定频道。

**代码/报错诊断**  
直接把报错贴过去：「帮我看看这个错误：TypeError: Cannot read property 'map' of undefined at...」或发一段代码问「这段代码有什么问题？」它会给出原因、修复建议或改好的代码。

**夜间自动化（多 Agent 并行）**  
社区里有人配置 7 个子 Agent 夜间并行：凌晨做基础设施健康检查、跑测试、备份数据、检查任务完成情况、API 监控等，早上起来看汇总结果即可；还有「每晚自动做安全审查，发现风险就告警，回复 fix it 就自动修」的用法。

### 内容与资讯

**医学通讯转播客**  
医生每天收到大量医学 Newsletter，没时间看。有人用 OpenClaw：自动拆邮件里的每条故事、抓取链接背景、写成对话式播客脚本，用 ElevenLabs 合成语音，超长就自动分段用 ffmpeg 拼接，通过 Signal 发送，通勤时听即可。

**奥运会/体育每日简报**  
设定每天早上 7 点自动整合：今日选手赛程、昨日奖牌、热门话题，发到 Telegram 体育频道。有用户用来跟意大利奥运代表队，AI 在主流媒体报道前就抓到爆料（如破纪录、伤病、争议等）。

**社交媒体 + 晨间简报**  
安排每天凌晨爬取技术/金融/娱乐等方向的热门（如小红书、微博、RSS），用大模型做摘要，早上通过飞书/钉钉推送到群。文案或自媒体也可以用来追踪各平台数据，自动生成每日简报。

### 企业/金融（落地案例）

**企业财务**  
报销：发票 OCR、自动填表、审核规则；数据处理从 T+1 到 T+0，人力成本明显下降；报销周期从几天缩短到 1 小时内，审核人力减少约 70%。  
报表：自动生成可视化报表、识别发票、处理报销流程。

**投研与风控**  
投研：3 分钟内出 A 股/美股专业投研报告，自动财务分析、估值建模、DCF；有的还做带风控的量化交易智能体。  
风控：欺诈识别准确率提升（如从 92% 到 98%+）、实时交易监控、异常模式识别、风控日志审计。

### 浏览器与自动化

**网页签到**  
「每天早上 9 点帮我签到这个网站，用户名 xxx 密码 xxx」。装 Browser Skill 后，到点自动打开浏览器、登录、完成签到并反馈结果。

**价格监控**  
「监控这个商品链接，价格低于 500 元就通知我」。定期检查价格，变动时推送，有的还会做价格历史趋势。

**定期抓数 + 报表**  
「每周一抓取这个页面的访问量、用户数、转化率，生成 Excel 发给我」。适合固定格式的运营/产品数据汇总。

### 社区与客服

**Discord/群内 FAQ 助手**  
先让 OpenClaw「记住：OpenClaw 的安装命令是 npm install -g openclaw@latest」等；之后有人问「如何安装 OpenClaw？」它会按记忆回答，相当于一个常驻的问答机器人。

**内容审核**  
配置敏感词和规则后，有人发不当内容就自动删除并警告，可设白名单（如管理员不触发）。

**活动报名**  
「创建活动：OpenClaw 线上分享会，3 月 10 日晚上 8 点，Discord 语音」「我要报名」「查看报名人数」「导出报名名单」。用自然语言完成创建、报名、统计、导出。

以上例子多数需要安装对应 **Skill**（如 `clawhub install browser`、`clawhub install github`），并在 `openclaw skills config <skill>` 里填好 API Key 或账号。更多案例可看 [OpenClaw 使用案例（官方教程）](https://openclawgithub.cc/guide/examples/) 和社区整理的 [Awesome OpenClaw Usecases](https://github.com/EvoLinkAI/awesome-openclaw-usecases-moltbook)。

**小结**：OpenClaw 的本质是「**接入第三方平台（微信 / 飞书 / 钉钉等）+ 本地部署的智能体 + 真正替你执行任务**」——你在这些 App 里用语音或文字和智能体交流，智能体在你本机或服务器上帮你完成收件箱、邮件/日程、文件整理、浏览器自动化、代码辅助等规划好的事。适合想有一个能干活、数据又在自己手里的 AI 助手的场景；上面这些是别人已经真在用的用法，你可以先挑一个场景跑通，再慢慢扩展。
